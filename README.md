Задание

Необходимо разработать приложение, которое предоставляет REST API для работы с кошельками пользователей.
1. Изменение баланса кошелька
Endpoint:
POST /api/v1/wallets/<WALLET_UUID>/operation
Пример запроса:
{
  "operation_type": "DEPOSIT" or "WITHDRAW",
  "amount": 1000
}

2. Получение баланса кошелька
Endpoint:
GET /api/v1/wallets/<WALLET_UUID>
Возвращает текущий баланс.

Обязательные условия

1. Кодовая база
    • Код должен соответствовать стандарту PEP8.
    • Необходимы миграции для базы данных.
    • Использовать асинхронный веб-фреймворк (к примеру FastAPI)

2. Конкурентность
Уделите внимание корректной работе приложения в конкурентной среде: параллельные запросы, включая запросы на изменение баланса одного и того же кошелька, должны обрабатываться корректно.

3. Инфраструктура
    • Приложение должно запускаться в Docker-контейнере.
    • База данных — также в контейнере.
    • Вся система поднимается одной командой через docker-compose.

4. Тестирование
    • Эндпоинты должны быть покрыты тестами.
